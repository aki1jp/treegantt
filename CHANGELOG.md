# 変更履歴 (Changelog)

本ファイルは TreeGantt の**製品リリース履歴**を記録します。
形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/)、バージョニングは [セマンティックバージョニング](https://semver.org/lang/ja/) に従います。

設計の変更点・要点は設計書 `docs/treegantt_design.md` の「改訂履歴」に記録します（役割分担: 製品リリース=本ファイル / 設計=設計書）。

## [1.1.0] - 2026-06-17

### 追加
- **プロジェクトごとのアドレス（URL）**: `/p/<プロジェクト名またはID>` で各プロジェクトを直接開けるようにした。名前でも ID でも開け、同名衝突時は ID で一意に解決。プロジェクト切替でアドレス欄が追従し、ブラウザの戻る/進む・ブックマーク・共有に対応。トップ `/` は従来どおり（前回プロジェクト／先頭）。

### 変更
- **作成ドラッグの開始閾値**: 日付未設定タスクをガント上でドラッグして日付を作成する操作に開始閾値（既定 4px）を導入。クリック時の手ぶれで日付が誤作成されるのを防ぎ、一定距離ドラッグして初めて作成される。

### 修正
- **プロジェクト切替時の白画面**: 行仮想化で前プロジェクトのスクロール位置が残ると可視範囲が空になり画面が真っ白になることがあった問題を修正（スクロール位置のクランプ＋切替時の先頭リセット）。

## [1.0.1] - 2026-06

### 修正
- 空ボディの DELETE が 400 で失敗し、タスク／プロジェクトの削除が効かない不具合を修正。
- CORS の許可メソッドを明示し、クロスオリジンの PATCH/DELETE がプリフライトで弾かれる不具合を修正。

### その他
- 本番配線のテスト化（`/health` の version・CORS・エラーハンドラ・WS ブロードキャスト）とフロントのユニット／カバレッジ計測を拡充。

## [1.0.0] - 2026-06-13

- TreeGantt 1.0 リリース。プロジェクト／タスクの CRUD、ツリー（WBS）＋ガントの統合ビュー、親サマリー・進捗集計、マイルストーン、依存関係（先行/後続）と矢印、クリティカルパス、リソースビュー、ドラッグ操作（移動／リサイズ／並べ替え／コピー＆挿入）、Markdown、CSV/JSON インポート・エクスポート、WebSocket によるリアルタイム同期、バージョン表示（ハンバーガー／`/health`）。
